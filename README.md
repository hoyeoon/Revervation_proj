## 프로젝트 소개
- 네이버 부스트코스 웹 백엔드 강좌를 수강하며 네이버 예약 서비스와 유사한 기능을 구현하는 프로젝트를 진행했습니다.
- 프론트엔드 개발자에게 제공하기 위한 Restful API 명세 개발을 했습니다.
## 기간
- 21.01.25 ~ 21.03.02
## 기술 스택
- Backend : **Spring 5, Java 8**
- RDBMS : **MySQL 8**
- Tool : **Eclipse, MySQL Workbench**
- 단위 테스트 : **Junit4 + Mockito**
- 통합 테스트 : **MockMVC**
- **Restful API**
